# MERN Jobify Portal

A job portal project built with a modern full-stack setup.

## What it is
A full-stack application for posting, searching, and managing jobs. It allows employers and job seekers to interact through a simple dashboard with job creation, editing, deletion, and status tracking.

## What it does
Users can register and log in securely, create and update job listings, delete jobs when no longer needed, and view summary statistics for jobs and application status. The app also includes search and filtering to find relevant jobs faster.

## Why it is useful
This project demonstrates a complete real-world application flow: frontend UI, backend API, database storage, authentication, and data management. It is useful as a portfolio project and as a foundation for learning how full-stack web applications work.

## Tech used
- Frontend: React with Vite
- Backend: Node.js and Express
- Database: MySQL
- Authentication: JWT
- Styling: CSS
